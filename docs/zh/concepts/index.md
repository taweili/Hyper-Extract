# Concepts

Hyper-Extract 采用**三层架构**：

- **Auto-Types** 定义了知识提取的数据结构
- **Methods** 基于 Auto-Types 提供提取算法
- **Templates** 提供领域特定的配置，包含开箱即用的 prompt

## Auto-Types

Auto-Types 是基础层 - 它们定义了知识提取的输出数据结构。
