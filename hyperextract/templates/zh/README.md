# Hyper-Extract 领域知识模板 (中文版)

欢迎使用 **Hyper-Extract 领域模板库**。这里提供了一系列针对中文语境和国内行业标准优化的提取模板。

## 设计理念
- **直觉导航**：采用扁平化目录结构，拒绝深层嵌套。
- **行业深度**：每个模板均由特定领域的知识结构（Schema）和提示词（Prompt）驱动。
- **文化融合**：在保持与国际标准接轨的同时，特别加入了如中医（TCM）、武侠（Wuxia）等中国特色文化领域。

## 领域索引

| 目录名称 | 说明 | 应用场景示例 |
| :--- | :--- | :--- |
| **`general`** | 通用百科 | 人物、机构、事件、地点等基础实体的快速抽取。 |
| **`finance`** | 金融投资 | A股/港股研报、基金招募说明书、上市公司季报。 |
| **`legal`** | 法律合规 | 合同关键要素（义务、权利）、案情调查、法条引用。 |
| **`education`** | 教育学术 | K12知识点图谱、课程大纲依赖关系、学术论文脉络。 |
| **`medicine`** | 医学健康 | 疾病症状映射、临床诊疗路径、西药说明书结构化。 |
| **`food`** | 美食餐饮 | 中餐菜谱（刀工、火候）、食材配比、八大菜系特征。 |
| **`history`** | 历史文化 | 朝代更迭时间线、重大战役、历史人物生平关系。 |
| **`literature`** | 文学 | 小说人物关系网、文学流派演变、古诗词意象关联。 |
| **`game`** | 游戏娱乐 | 游戏世界观设定、任务流向树、物品合成与掉落表。 |
| **`agriculture`** | 农业与养殖 | 作物生长周期、病虫害防治、畜牧谱系与饲养标准。 |
| **`biology`** | 生物科学 | 生物分类学、蛋白质相互作用、基因调控网络。 |
| **`chemistry`** | 化学科学 | 分子结构、化学反应方程式路径、材料物化特性。 |
| **`environment`** | 环境与气候 | 气象变化趋势、碳排放追踪、生态系统食物网。 |
| **`movie`** | 影视 | 剧本节拍器拆解、演员与幕后人员网络、影评情感。 |
| **`news`** | 新闻传媒 | 事件因果链、新闻专题聚类、深度报道事实板。 |
| **`tcm`** | **中医 (特色)** | 中草药方剂、经络穴位、辨证施治过程等。 |
| **`wuxia`** | **武侠/修真 (特色)** | 门派恩怨、功法境界、修仙等级体系、法宝神兵。 |

*(注：英文版对应的 15 个通用领域在中文版中均有映射文件夹，中医与武侠为中文版独有)*
