# Hyper-Extract 领域知识模板 (中文版)

欢迎使用 **Hyper-Extract 领域模板库**。这里提供了一系列针对中文语境和国内行业标准优化的提取模板。

## 设计理念
- **直觉导航**：采用扁平化目录结构，拒绝深层嵌套。
- **行业深度**：每个模板均由特定领域的知识结构（Schema）和提示词（Prompt）驱动。
- **文化融合**：在保持与国际标准接轨的同时，特别加入了如中医（TCM）、武侠（Wuxia）等中国特色文化领域。
