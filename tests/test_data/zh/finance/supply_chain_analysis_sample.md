# 新能源汽车供应链风险评估报告

**编制单位**：德勤管理咨询（上海）有限公司
**委托方**：华锐新能源汽车科技股份有限公司
**报告日期**：2025年11月
**密级**：机密——仅限委托方内部使用

---

## 一、报告摘要

本报告对华锐新能源汽车科技股份有限公司（以下简称"华锐"或"公司"）的核心零部件供应链进行了全面的风险评估。评估范围覆盖公司38家一级供应商、96家二级供应商以及关键原材料供应源，涉及12个国家和地区。报告识别了6个高风险供应链节点，并针对每个风险提出了缓解建议。

### 2.2 二级供应商关键依赖

**锂电池关键原材料**

- **天齐锂业**（四川射洪）→ 向宁德时代供应电池级碳酸锂。天齐锂业控制着全球约10%的锂矿资源，其澳洲Greenbushes矿是全球最大的硬岩锂矿。碳酸锂价格自2022年高点人民币60万元/吨已下跌至约8万元/吨，但供给端仍存在集中度风险。

- **华友钴业**（浙江桐乡）→ 向宁德时代和中创新航供应电池级硫酸钴和前驱体。钴资源70%来自刚果（金），存在地缘政治风险和ESG合规风险。

- **格林美**（湖北荆门）→ 电池回收与前驱体材料。公司是国内最大的动力电池回收企业，回收产能达30万吨/年，有助于降低对上游矿产资源的依赖。

**半导体制造链**

- **台积电（TSMC）**（台湾新竹）→ 为英伟达和高通代工制造智能驾驶/座舱芯片。英伟达Orin X芯片采用三星8nm工艺生产，但下一代Thor芯片将转至台积电5nm。高通8295采用台积电5nm工艺。台积电是智能驾驶芯片供应链中无可替代的关键节点。

- **日月光（ASE）**（台湾高雄）→ 为英伟达和高通提供先进封装服务。封装环节是当前芯片供应链中产能最为紧张的环节之一。

**稀土永磁材料**

- **北方稀土**（内蒙古包头）→ 向汇川技术和日本电产供应钕铁硼永磁材料，用于制造驱动电机中的永磁体。中国控制全球约60%的稀土矿产和90%的稀土加工产能。2025年以来，稀土出口管制政策有所收紧。

## 三、风险评估

### 3.1 地缘政治风险 — 极高

**中美科技竞争与芯片管制**

华锐供应链面临的最大地缘政治风险来自于中美科技脱钩。公司对美系芯片的依赖程度较高：

- 英伟达Orin X智能驾驶芯片：占公司高阶智能驾驶方案的100%芯片供应，年采购额人民币8.6亿元
- 高通8295座舱芯片：占公司中高端座舱方案的80%芯片供应，年采购额人民币4.8亿元

**风险评估**：美国商务部已于2022年和2023年多次升级对华芯片出口管制。虽然当前管制主要针对先进AI训练芯片，但未来不排除将汽车用高算力芯片纳入管制清单的可能性。一旦英伟达和高通芯片被禁止向中国出口，公司高阶智能驾驶和座舱系统的生产将在现有库存耗尽后（约4-6个月）面临停产风险。

**缓解措施**：
1. 加速地平线征程6芯片的适配验证工作，计划2026年Q3完成全部车型适配（预计投入人民币1.2亿元）
2. 与华为MDC智能驾驶平台启动合作评估
3. 将英伟达和高通芯片的战略储备从当前的4个月提升至8个月（需增加约人民币6.7亿元库存资金）

**台湾海峡风险**

台积电作为英伟达和高通芯片的核心代工方，是供应链中的关键瓶颈。台海局势的任何重大变化都可能导致全球芯片供应中断。

**风险评估**：台湾相关供应链中断将影响公司约35%的零部件供应（按采购额计算），涉及智能驾驶芯片、座舱芯片、部分电子元器件。短期内不存在替代方案。

### 3.2 集中度风险 — 高

**单一来源依赖**

公司存在5项无替代供应商的关键零部件：

1. **英伟达Orin X芯片**：高阶智能驾驶唯一芯片方案（地平线征程6适配中，预计2026年H2可替代）
2. **高通8295座舱芯片**：中高端座舱唯一芯片方案（联发科MT8678适配中，预计2026年Q4可替代）
3. **宁德时代CTP 3.0电池包**：特定车型专用电池包结构设计（其他电池厂的CTP方案需6-9个月重新开模）
4. **博世iBooster制动系统**：线控制动唯一供应商（比亚迪e-Booster方案验证中）
5. **日本电产E-Axle**：高端车型后电驱总成（汇川E-Axle方案预计2026年Q2完成验证）

**风险评估**：基于历史数据，任一供应商在一年内发生重大供应中断（超过4周）的概率约为12%-15%。

### 3.3 原材料价格波动风险 — 中

**关键原材料价格走势**

| 原材料 | 当前价格 | 12个月前价格 | 变动 | 对公司成本影响 |
|---|---|---|---|---|
| 碳酸锂 | 8.2万元/吨 | 10.5万元/吨 | -21.9% | 每万元/吨变动影响电池成本约1.2% |
| 钴 | 20.5万元/吨 | 23.8万元/吨 | -13.9% | 影响三元电池成本约0.4% |
| 镍 | 12.8万元/吨 | 14.2万元/吨 | -9.9% | 影响三元电池成本约0.5% |
| 铜 | 7.2万元/吨 | 6.8万元/吨 | +5.9% | 影响电驱和线束成本约0.3% |
| 稀土（钕） | 52万元/吨 | 48万元/吨 | +8.3% | 影响电机成本约0.6% |

当前原材料价格整体处于近两年低位，有利于公司控制成本。但碳酸锂价格已接近部分锂矿企业的成本线（约6-7万元/吨），进一步下跌空间有限，反弹风险正在积累。

### 3.4 自然灾害与极端事件风险 — 中

**地震风险**：日本电产京都工厂和台积电新竹工厂位于地震活跃带。2024年1月能登半岛地震曾导致日本电产部分产线停工2周。

**极端天气**：宁德时代宁德工厂和中创新航常州工厂分别位于台风影响区域和洪涝易发区域。2024年台风"贝碧嘉"导致宁德地区工业用电短暂中断。

**流行病**：新冠疫情期间的供应链中断经验表明，集中在单一地区的供应链面临较大的流行病风险。目前公司已建立一定的多地备份能力。

## 四、供应链网络拓扑

### 4.1 关键路径分析

**路径一：智能驾驶系统供应链**（风险等级：极高）

```
台积电（台湾）→ 芯片制造
  → 日月光（台湾）→ 先进封装
    → 英伟达（美国设计）→ Orin X芯片
      → 德赛西威（中国惠州）→ 域控制器组装
        → 华锐（中国合肥）→ 整车集成
```

该路径涉及3个国家/地区，包含美国出口管制风险和台海风险，全链条无替代方案。端到端交付周期：20-26周。

**路径二：动力电池供应链**（风险等级：中）

```
天齐锂业/赣锋锂业（中国）→ 碳酸锂/氢氧化锂
华友钴业（中国）→ 钴前驱体
  → 宁德时代（中国宁德）→ 电芯制造 → Pack组装
    → 华锐（中国合肥）→ 整车集成
```

该路径主要在国内闭环，供应链安全性较高。但宁德时代的产能集中在福建和四川，区域集中度较高。端到端交付周期：8-12周。

**路径三：电驱动系统供应链**（风险等级：中-高）

```
北方稀土（中国内蒙古）→ 钕铁硼永磁材料
  → 日本电产（日本京都）→ 驱动电机制造
    → 华锐（中国合肥）→ 整车集成
```

该路径依赖日本供应商，存在地震风险和中日关系波动风险。端到端交付周期：14-18周。

## 六、建议

1. **立即行动（0-6个月）**：增加英伟达、高通芯片战略库存至8个月用量；启动地平线征程6、联发科MT8678的量产适配工作。

2. **短期（6-18个月）**：完成国产智能驾驶芯片的全车型适配和量产导入；建立供应链实时监控平台，覆盖一级和二级供应商。

3. **中期（18-36个月）**：实现关键零部件"双供应商"策略全覆盖，消除所有单一来源风险；在东南亚（越南/泰国）建立备份零部件仓储中心。

4. **长期（持续）**：参与国家"强链补链"专项行动，支持国产芯片和核心零部件生态发展；建立供应商ESG评估体系，防范合规风险。
