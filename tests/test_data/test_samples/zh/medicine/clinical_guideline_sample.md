# 2型糖尿病防治指南（2020版）

## 一、概述

2型糖尿病（T2DM）是一种常见的慢性代谢性疾病，其主要特征是胰岛素抵抗和胰岛素分泌不足，导致血糖水平升高。本指南旨在为临床医生提供规范化的诊疗建议，以提高2型糖尿病的防治水平。

## 二、诊断标准

### 2.1 糖尿病诊断标准
- 糖化血红蛋白（HbA1c）≥6.5%
- 空腹血糖（FPG）≥7.0mmol/L
- 口服葡萄糖耐量试验（OGTT）2小时血糖≥11.1mmol/L
- 随机血糖≥11.1mmol/L且伴有典型糖尿病症状

### 2.2 糖尿病前期诊断标准
- 空腹血糖受损（IFG）：6.1≤FPG<7.0mmol/L
- 糖耐量减低（IGT）：7.8≤OGTT 2小时血糖<11.1mmol/L
- 糖化血红蛋白：5.7%≤HbA1c<6.5%

## 三、治疗路径

### 3.1 生活方式干预
**推荐级别**：A
**证据等级**：1a

所有2型糖尿病患者都应接受生活方式干预，包括：
- 合理饮食：控制总热量，合理分配碳水化合物、蛋白质和脂肪
- 规律运动：每周至少150分钟中等强度有氧运动
- 控制体重：将BMI控制在24kg/m²以下
- 戒烟限酒
- 心理平衡

### 3.2 血糖控制目标
**推荐级别**：A
**证据等级**：1a

- 一般成人：HbA1c<7.0%
- 老年患者（65岁以上）：HbA1c<7.5%（健康状况良好）或<8.0%（健康状况较差）
- 妊娠糖尿病：HbA1c<6.5%
- 有严重并发症或低血糖风险高的患者：HbA1c<8.0%

### 3.3 药物治疗路径

#### 3.3.1 单药治疗
**推荐级别**：A
**证据等级**：1a

- 首选二甲双胍：起始剂量500mg，每日2-3次，最大剂量2000mg/日
- 若二甲双胍不耐受，可选择：
  - 磺脲类：格列美脲1-4mg/日
  - 噻唑烷二酮类：吡格列酮15-45mg/日
  - DPP-4抑制剂：西格列汀100mg/日
  - SGLT-2抑制剂：达格列净10mg/日

#### 3.3.2 二联治疗
**推荐级别**：A
**证据等级**：1a

若单药治疗3个月后HbA1c未达标，应启动二联治疗：
- 二甲双胍 + 磺脲类
- 二甲双胍 + 噻唑烷二酮类
- 二甲双胍 + DPP-4抑制剂
- 二甲双胍 + SGLT-2抑制剂
- 二甲双胍 + GLP-1受体激动剂

#### 3.3.3 三联治疗
**推荐级别**：A
**证据等级**：1b

若二联治疗3个月后HbA1c未达标，应启动三联治疗：
- 二甲双胍 + 磺脲类 + 噻唑烷二酮类
- 二甲双胍 + 磺脲类 + DPP-4抑制剂
- 二甲双胍 + 磺脲类 + SGLT-2抑制剂
- 二甲双胍 + 磺脲类 + GLP-1受体激动剂

#### 3.3.4 胰岛素治疗
**推荐级别**：A
**证据等级**：1a

若三联治疗3个月后HbA1c未达标，应启动胰岛素治疗：
- 基础胰岛素：甘精胰岛素，起始剂量0.1-0.2U/kg/日
- 预混胰岛素：门冬胰岛素30，起始剂量0.2-0.4U/kg/日，早晚餐前注射
- 餐时+基础胰岛素：门冬胰岛素+甘精胰岛素，根据血糖调整剂量

## 四、并发症筛查与管理

### 4.1 大血管并发症
**推荐级别**：A
**证据等级**：1a

- 心血管疾病：每年进行一次心电图检查，评估心血管风险
- 脑血管疾病：定期进行颈部血管超声检查
- 下肢血管疾病：每年进行一次踝肱指数（ABI）检查

### 4.2 微血管并发症
**推荐级别**：A
**证据等级**：1a

- 糖尿病肾病：每年进行一次尿白蛋白/肌酐比值（UACR）和血肌酐检查
- 糖尿病视网膜病变：每年进行一次眼底检查
- 糖尿病神经病变：每年进行一次足部感觉检查

## 五、随访与监测

### 5.1 随访频率
- 新诊断患者：每2-4周随访一次，直至血糖达标
- 血糖控制稳定患者：每3-6个月随访一次
- 有并发症患者：根据并发症严重程度增加随访频率

### 5.2 监测指标
- 血糖：空腹血糖、餐后2小时血糖
- 糖化血红蛋白：每3个月检测一次
- 血脂：每6个月检测一次
- 血压：每次随访时检测
- 体重：每次随访时检测
- 足部检查：每3个月进行一次

## 六、特殊人群管理

### 6.1 老年糖尿病患者
**推荐级别**：B
**证据等级**：2a

- 血糖控制目标应适当放宽，避免低血糖
- 优先考虑低血糖风险低的药物
- 综合评估心肾功能，调整药物剂量

### 6.2 妊娠糖尿病患者
**推荐级别**：A
**证据等级**：1a

- 严格控制血糖，目标：空腹血糖<5.3mmol/L，餐后1小时血糖<7.8mmol/L，餐后2小时血糖<6.7mmol/L
- 首选胰岛素治疗
- 产后6-12周进行OGTT复查

### 6.3 合并心血管疾病患者
**推荐级别**：A
**证据等级**：1a

- 优先选择有心血管获益的药物：SGLT-2抑制剂、GLP-1受体激动剂
- 血压控制目标：<130/80mmHg
- 血脂控制目标：LDL-C<1.8mmol/L
