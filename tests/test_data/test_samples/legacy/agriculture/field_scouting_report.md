## 总体病虫害发生趋势

根据本次巡田调查，当前水稻病虫害总体呈轻度发生态势。分蘖期主要病虫害以稻飞虱和稻纵卷叶螟为主，大部分田块发生较轻，少数田块需要密切关注。

### 需要重点关注的田块

1. **D-002**：生长偏旺，需及时晒田控苗
2. **D-003**：纹枯病开始发生，需及时防治
3. **D-004**：出现僵苗，需紧急处理

### 下一阶段管理建议

1. 及时晒田：D-001、D-002、D-003应在3天内开始晒田
2. 病虫害防治：重点关注D-003的纹枯病防治
3. 促弱转壮：D-004需加强水肥管理，促进恢复
4. 监测预警：密切关注稻飞虱和稻纵卷叶螟的发生动态
