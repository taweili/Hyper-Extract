# 红烧肉标准食谱

## 基本信息

- **菜名**：红烧肉
- **菜系**：浙菜/本帮菜
- **分类**：热菜 - 肉类
- **口味**：咸甜鲜香
- **难度**：中等
- **预估烹饪时间**：90分钟
- **适合人数**：4-6人份

## 烹饪步骤

### 第一步：预处理
1. 将五花肉切成约3-4厘米见方的块状
2. 生姜切片，大葱切段备用
3. 冷水下锅，加入料酒和姜片，放入五花肉块
4. 大火烧开后撇去浮沫，煮约3分钟后捞出，用温水洗净备用

### 第二步：炒糖色
1. 锅中放入少量油，小火加热
2. 放入冰糖，慢慢搅拌至冰糖融化
3. 待糖色变成琥珀色并起小泡时，迅速放入五花肉翻炒
4. 翻炒至每块肉都均匀裹上糖色

### 第三步：炖煮
1. 加入姜片、葱段、八角、桂皮、香叶、干辣椒
2. 倒入生抽、老抽、料酒，翻炒均匀
3. 加入800毫升清水，大火烧开后转小火
4. 盖上锅盖，小火慢炖约60分钟

### 第四步：收汁
1. 炖至肉酥烂时，开大火收汁
2. 不断翻动肉块，使其均匀上色
3. 待汤汁浓稠至包裹肉块时，加入适量盐和鸡精调味
4. 出锅装盘，撒上葱花即可

---

## 关键控制点（TIPS）

1. **选材**：必须选用肥瘦相间的五花肉，肥瘦比例约为3:7，口感最佳
2. **焯水**：冷水下锅可以更好地去除血水和腥味
3. **炒糖色**：糖色炒至琥珀色即可，千万别炒焦，否则会发苦
4. **火候**：炖煮时一定要用小火，急火会使肉质变柴
5. **收汁**：最后收汁时要不断翻动，避免糊锅
6. **保存**：红烧肉放凉后放入冰箱冷藏，可保存3-5天

## 变体与创新

- **本帮红烧肉**：加入上海黄酒，甜度稍高
- **毛氏红烧肉**：不加酱油，保持肉的原色
- **家常红烧肉**：可加入土豆、鹌鹑蛋等配菜
